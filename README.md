# FESB-PROJECT

Instalirati na svom mobitelu aplikaciju imena "Expo Go" te kada se pokrene naredba "npx expo start" onda skenirati s toom aplikacijom QR-code.

## Project Setup

```sh
npm install
```

### For Compile

```sh
expo install
```

### Compile and Hot-Reload for Development

```sh
npx expo start
```
